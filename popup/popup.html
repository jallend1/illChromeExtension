Keyboard Shortcuts

Use the following keyboard shortcuts to quickly access ILL tools:

  • Ctrl+Shift+1

    Copies the request data from the open WorldShare tab

    Open borrowing request in WorldShare

  • Ctrl+Shift+2

    Pastes the request data into Evergreen

    Enter New ILL screen in Evergreen

  • Ctrl+Shift+3

    Generates Overdue Letter

    Patron checked out items screen

  • Ctrl+Shift+Y

    Copies Library Address from WorldShare Request (For Dymo Printing)

    Open lending request in WorldShare
